# devops-project
Этот проект показывает, как я упаковал своё Java‑приложение в Docker‑образ и развернул его в Kubernetes‑кластере. Основная идея была в том, чтобы продемонстрировать полный цикл: от сборки JAR‑файла до запуска приложения в управляемой среде.
Для контейнеризации я использовал Docker — приложение собрано в образ, что позволяет легко переносить его и запускать в одинаковом окружении.
Для оркестрации я применил Kubernetes: через Deployment я задал количество реплик и стратегию обновления, а кластер сам следит за состоянием подов и перезапускает их при сбоях.
