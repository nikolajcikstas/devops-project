# devops-project
Простое Java‑приложение, упакованное в Docker‑контейнер и развёрнутое с помощью Kubernetes.
В процессе работы были использованы базовые команды Docker для сборки и запуска образа, а также манифесты Kubernetes для управления контейнерами и их масштабирования. 
Репозиторий создан и загружен в GitHub с использованием Git, что позволяет легко отслеживать изменения и демонстрировать проект как пример практического применения DevOps‑инструментов.
